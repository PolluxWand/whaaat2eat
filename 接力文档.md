# "选了么" 项目接力文档

## 🎯 项目概述

### 项目简介
**"选了么 - 今天吃什么"** 是一个帮助用户解决"选择困难症"的网页应用。通过转盘抽取的方式，随机推荐美食，让用户快速决定吃什么。

### 核心功能
1. **转盘抽取** - 旋转转盘随机选择美食
2. **分类筛选** - 按快餐/正餐/甜品等分类浏览
3. **收藏系统** - 收藏喜欢的美食，支持权重优先
4. **历史记录** - 查看最近选择的美食
5. **设置面板** - 
   - 口味屏蔽（不吃辣、不吃生冷等）
   - 菜系屏蔽（排除特定菜系）
   - 时间感知（早中晚推荐不同品类）
6. **主题切换** - 深色/浅色模式
7. **海报生成** - 生成美食卡片可分享

### 技术栈
- **纯前端单文件** - 只有一个 `index.html` 文件
- **React 18.2.0** - UI 框架
- **Tailwind CSS CDN** - 样式
- **Babel Standalone** - 浏览器端 JSX 转译
- **html2canvas** - 海报生成

### 设计理念
- 📱 **移动优先** - 主要面向手机端使用
- 🎨 **简洁直观** - 操作简单，无学习成本
- 📦 **单文件部署** - 无需构建，直接部署 HTML
- 🎵 **音效反馈** - 转盘滚动、停止都有声音

## 📝 备注

### 设计约束
1. **单 HTML 文件** - 不能拆分成多个文件
2. **无构建步骤** - 不能用 npm、Vite 等工具
3. **浏览器端运行** - 所有代码在浏览器执行
4. **移动优先** - 主要面向手机用户

1. **性能** - 浏览器端 Babel 转译较慢
